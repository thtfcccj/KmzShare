## ●标准与规则

### ■重要约定：
 * “n”表示十进制数字,如：1 ，15
 * “N”表示汉字数字，如：一， 十二
 * “xx”表示名称，如国博

### ■管理器层次结构：
#### ◆一层：  区分阶段，以阶段(重要注解)命名：   
  * 运营段(截止日期）      (以下简称运营)
  * 在建段(N期)               (以下简称在建)
  * 近期规划(N期N轮)       (以下简称近期)
  * 远期规划                   (以下简称远期)
  * 参考与预览               存放中间文档  (以下简称附件)
#### ◆二层：  区分：
  * “走向”即线路
  * “站点”即站位置与名称
  * “站点分布”仅含站点图标的预览，并用颜色区分阶段：
     + 运营： 绿色
     + 在建： 黄色
     + 规划：粉色
     制作方法为：复制(制件好的)站点为"站点分布"->其上点属性->样式栏点共享样式->
                       ** 标签改为透明，图标改为0.6及对应颜色，去掉“允许展天此文件夹”选项 ** 。
#### ◆三层：  以“n号线”+“制式”+ "_子线(多条时有，如正线或xx线)或工程(非运行时)名称"命名，如：
  * 1号线地铁
  * 2号线单轨
  * 15号线城轨快线
  * 3号线单轨_正线(子线名)
  * 4号线地铁_西延(工程名)
#### ◆四层：同一线路中，区分重要阶段时可能存在此情况(部分为做图的历史原因，未进行合并)，如：
  * 2号线单轨中的：一二期和南延段

### ■线路属性：
   以可辨识不同线路，不周阶段为原则制定：
     * 颜色：官方指定色以方便辨识。(官网未找到，百度百科里的配色图)
     * 宽度：区分运营，在建与近期，若为快线(含后期市郊线)+0.5, 分别为
      + 运营：4.0
      + 在建：3.0
      + 近期：2.0 ,快轨+0.5 = 2.5

### ■站点属性：
  以匹配线路属性为原则(如：线由近到远越来越细，则标签与图标也会越来越小)
#### ◆标签：
     * 颜色：同线路，但若显示不清，可适当调亮！！！
     * 比例：区分运营，在建与近期，若为快线(含后期市郊线)+0.25, 分别为
      + 运营：0.7
      + 在建：0.65
      + 近期：0.6 ,快轨+0.25 = 0.625
#### ◆图标：
     * 颜色：同线路
     * 比例：区分运营，在建与近期，若为快线(含后期市郊线)+0.1, 分别为
      + 运营：1.0
      + 在建：0.8
      + 近期：0.6 ,快轨+0.1 = 0.7

### ■关于官方未指定色的定义：
  主要指18号及以上线路，暂定为：
  * 18号线:  RGB=170,0,127 
  * 24号线：RGB=255,127,255
